Set the type of chart using the chartType property. Available options:

"range" – for range plot
"dot" – for dot plots
"arrow" – for arrow plot
"bar" – for bar charts with a reference point

showGuidelines Option

Controls whether guidelines are displayed on the chart. Guidelines help users read the chart when there's a lot of white space.

For "range" and "arrow" charts: Guidelines can be toggled on or off.
For dot plot: Guidelines are always on.
For bar charts with reference markers: Guidelines are always off.

Data Labels

Data labels can be enabled or disabled depending on the chart type:

For "range", "dot", and "arrow" charts:
You can turn data labels on or off, and optionally restrict them to desktop view using the desktopOnly setting.

For "bar" charts with reference markers:
Data labels are not supported
